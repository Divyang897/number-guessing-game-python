# Number Guessing Game

This project is a simple number guessing game where the user tries to guess
a randomly generated number.

## Features
- Random number generation
- User feedback (Too High / Too Low)
- Displays number of attempts

## Technology Used
- Python

## How to Run
python number_guessing.py
